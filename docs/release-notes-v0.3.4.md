# Release Notes — v0.3.4

## Summary

This release continues the command-center dashboard refinement work with a stronger Chinese-first overview experience.

## Highlights

- Home page now reliably stays on the **overview landing page**
- Clicking the brand title returns directly to the homepage overview
- More team/task/role text is translated into **Simplified Chinese** at the display layer
- The dashboard now feels more consistent for Chinese-language operational use

## What changed

### UX behavior
- Removed the old auto-open behavior that jumped into a single team when only one team existed
- Kept the overview page as the default landing page
- Added a direct “return home” interaction through the top-left title area

### Localization
- Team descriptions are now shown in more natural Simplified Chinese
- Member role labels are localized for display
- Task subject patterns are translated into Chinese-friendly summaries for dashboard viewing

## Practical impact

This release improves readability and operational clarity for Chinese-speaking users without changing the underlying ClawTeam task data model.
